# Séquence Email — Pilier Conscient

## RÈGLES UNIVERSELLES D'EMAIL PILIER CONSCIENT

- **Objet** : 6-8 mots, curiosité + bénéfice concret
- **Première ligne** : accroche directe — pas juste "Bonjour [Prénom]"
- **Corps** : 150-300 mots, 1 idée centrale, 1 histoire courte ou vérité, 1 CTA
- **Signature** : "Avec respect, Prince Johann | Pilier Conscient"
- **Fréquence** : max 3-4 emails/semaine en séquence chaude, 1-2/semaine en nurturing

---

## SÉQUENCES TYPE

### SÉQUENCE 1 — Après Quiz (Jours 1-7)

```
EMAIL J0 (immédiat après quiz) :
Objet : Tes résultats — et ce qu'ils révèlent
Corps : Résultats personnalisés par score → pilier le plus faible → invitation à aller plus loin

EMAIL J2 :
Objet : Ce que [pilier faible] coûte vraiment à un homme
Corps : Histoire d'homme avec ce pilier faible → coût invisible → solution du Code

EMAIL J4 :
Objet : La 1 chose à faire cette semaine
Corps : Action concrète liée au pilier faible → protocole 7 jours → CTA programme

EMAIL J6 :
Objet : Est-ce que tu t'es reconnu ?
Corps : Question directe → témoignage transformateur → invitation au programme
```

---

### SÉQUENCE 2 — Après Achat Programme (Jours 1-14)

```
EMAIL J0 (confirmation) :
Objet : Bienvenue dans le Code, [Prénom]
Corps : Accueil solennel → ce qui les attend → comment accéder → 1 action immédiate (Module 1)

EMAIL J1 :
Objet : Avant de commencer — lis ça
Corps : Le contrat avec soi-même → comment tirer le maximum → la règle des 90 jours

EMAIL J3 :
Objet : Comment s'est passé le Module 1 ?
Corps : Question d'engagement → piliers de fondation → encouragement

EMAIL J7 :
Objet : 7 jours. Qu'est-ce qui a changé ?
Corps : Premier bilan → victoires à célébrer → comment continuer sur l'élan

EMAIL J14 :
Objet : La vérité sur la 2e semaine
Corps : La semaine 2 est plus difficile que la 1 → pourquoi → comment tenir
```

---

### SÉQUENCE 3 — Nurturing Long Terme (Hebdomadaire)

Envoyée chaque semaine à la liste complète. Contenu de valeur + CTA doux.

```
Format standard :
Objet : [Sujet du pilier de la semaine] — [bénéfice]
Corps : 
→ Accroche (2-3 lignes)
→ Histoire d'homme (4-5 lignes)
→ Leçon du Code (5-6 lignes)
→ Action de la semaine (2-3 lignes)
→ CTA : "Si tu veux aller plus loin → [lien quiz ou programme]"
```

---

### SÉQUENCE 4 — Relance Abandonnés (Jours 1-5)

Pour les prospects qui ont commencé le processus d'achat sans finaliser.

```
EMAIL J1 :
Objet : Tu as failli faire quelque chose d'important
Corps : Direct → ce qu'il a laissé → enjeu de ne pas agir → CTA retour

EMAIL J3 :
Objet : [Prénom], une dernière chose
Corps : Réponse à l'objection la plus probable → preuve sociale → CTA

EMAIL J5 :
Objet : C'est ok si ce n'est pas le bon moment
Corps : Respect de la décision → valeur offerte quand même → porte ouverte
```

---

## RÈGLES DE COMPORTEMENT

- **Voix Prince Johann** — appliquer `skill-voix-prince-johann`
- **1 CTA par email** — jamais 3 liens différents
- **Jamais d'urgence artificielle** — contraire à l'Authenticité (P11)
- **Personnaliser selon le segment** — un email pour les acheteurs ≠ un email pour les prospects
