# Module Programme — 52 Semaines Code Masculin

## STRUCTURE D'UN MODULE COMPLET

Chaque module contient 4 composants :

---

### COMPOSANT 1 — SCRIPT VIDÉO (15-25 min)

```
INTRO (1-2 min) :
"Bienvenue dans le Module [N] — [Titre].
Ce module fait partie du Pilier [X] : [Nom du pilier].
[1-2 phrases sur l'enjeu de ce module spécifiquement]"

HISTOIRE D'OUVERTURE (3-5 min) :
[Histoire d'homme réel — voir structure du script podcast Bloc 3]

CONCEPT PRINCIPAL (8-12 min) :
3-4 sous-points structurés :
→ [Concept 1] : [Explication + exemple concret]
→ [Concept 2] : [Explication + exemple concret]
→ [Concept 3] : [Explication + exemple concret]

EXERCICE PRATIQUE (3-5 min) :
"Cette semaine, voilà ce que tu vas faire : [Défi hebdomadaire détaillé]
Concrètement, ça ressemble à ça : [exemple d'application]"

AFFIRMATIONS (1 min) :
5 affirmations liées au pilier — lues lentement

VÉRITÉ FINALE (1-2 min) :
"L'homme du Code qui travaille ce pilier [X], il ressemble à ça : [description]
Tu sais maintenant ce que tu dois faire. Le reste n'appartient qu'à toi."
```

---

### COMPOSANT 2 — PDF GUIDE

```
EN-TÊTE : Logo Pilier Conscient | Module [N] | Pilier [X]

SECTION 1 — OBJECTIF DU MODULE
[Ce que l'homme va comprendre ou transformer cette semaine]

SECTION 2 — CONCEPTS CLÉS
Point 1 : [Titre] → [Explication 3-5 lignes]
Point 2 : [Titre] → [Explication 3-5 lignes]
Point 3 : [Titre] → [Explication 3-5 lignes]

SECTION 3 — DÉFI HEBDOMADAIRE
[Défi précis — qui / quoi / quand / comment]

SECTION 4 — JOURNAL DE RÉFLEXION
5-7 questions calibrées sur le pilier du module
(utiliser la banque de `skill-journal-reflexion-pilier`)

SECTION 5 — AFFIRMATIONS DE LA SEMAINE
5 affirmations numérotées — à répéter chaque matin

SECTION 6 — MON ENGAGEMENT
"Cette semaine, je m'engage à : _______________"
Signature : _______________ Date : _______________

PIED DE PAGE : pilierconscient.com | @pilierconscient
```

---

### COMPOSANT 3 — DÉFI HEBDOMADAIRE

Format standard :
```
🎯 DÉFI MODULE [N]

[Titre du défi — court et percutant]

CE QUE TU FAIS :
[Action concrète — mesurable, datée, précise]

COMMENT TU SAIS QUE TU L'AS ACCOMPLI :
[Critère objectif de succès]

POURQUOI CE DÉFI :
[Lien avec le pilier et la transformation visée en 2-3 lignes]

NIVEAU DE DIFFICULTÉ : ★★★☆☆ (adapter selon le module)
```

---

### COMPOSANT 4 — QUESTIONS JOURNAL

5 questions pour le journal de réflexion du module, calibrées sur le pilier.
Aller de la surface vers le fond (voir `skill-journal-reflexion-pilier` pour la banque complète).

---

## PROTOCOLE DE GÉNÉRATION

### Informations nécessaires

```
1. Numéro du module (1-52)
2. Titre du module
3. Pilier concerné (P1-P12)
4. Phase (Fondation / Identité / Impact)
5. Composants demandés (tous / sélection)
6. Niveau d'avancement de l'homme type (débutant / intermédiaire)
```

---

## RÈGLES DE COMPORTEMENT

- **Voix Prince Johann** — appliquer `skill-voix-prince-johann` sur le script vidéo et le PDF
- **1 seul concept central par module** — pas condensation de tout le pilier en 1 module
- **Le défi doit être légèrement inconfortable** — pas trivial, pas impossible
- **La vérité finale doit laisser l'homme debout** — pas écrasé, pas flatté : debout
