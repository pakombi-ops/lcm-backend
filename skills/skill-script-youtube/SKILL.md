# Script YouTube — Pilier Conscient

## STRUCTURE DE LA VIDÉO (8-15 min)

### BLOC 1 — HOOK (0-30 secondes)
Le hook doit créer une rupture immédiate. L'algorithme juge les 30 premières secondes.

**Formats de hook :**

**La question qui dérange :**
```
"[Question provocatrice liée au sujet] — et si la réponse que tu crois avoir est complètement fausse ?"
```

**L'affirmation contre-intuitive :**
```
"[Chose que tout le monde croit] — c'est précisément pour ça que [problème].
Dans cette vidéo, je te montre pourquoi."
```

**La statistique ou vérité froide :**
```
"[Fait ou vérité inconfortable sur le sujet].
Ce que je vais te dire là va peut-être te déranger.
Mais rester dans l'ignorance coûte plus cher."
```

**Règles du hook YouTube :**
- Pas de générique au début — on entre dans le vif immédiatement
- Créer une tension ou une promesse en moins de 30 secondes
- Pas de "abonnez-vous" avant d'avoir apporté de la valeur

---

### BLOC 2 — PROBLÈME (30s-2min)
Développer le problème que cette vidéo résout.

```
"Voilà ce que j'observe chez les hommes qui viennent me parler de [sujet] :
[Description du problème en 3 manifestations concrètes]

Ce n'est pas de la malveillance. Ce n'est pas de la faiblesse.
C'est [diagnostic du Code Masculin].

Et la bonne nouvelle : c'est un pilier qu'on peut travailler."
```

---

### BLOC 3 — ANALYSE CODE MASCULIN (3-8 min)
Le cœur de la vidéo. 3 sous-parties maximum.

**Format pour chaque sous-partie :**
```
[Titre court de la sous-partie — ex : "La première erreur"]

"[Explication du concept — direct, incarné, avec exemple concret]

Concrètement, ça ressemble à ça dans ta vie : [exemple du quotidien]

Ce que le Pilier [X] nous dit là-dessus : [lien avec le Code]"
```

**Format réaction (si la vidéo analyse un contenu externe) :**
```
"[Description de ce que l'autre dit]
Ce que j'observe là : [analyse Prince Johann]
Ce que le Code Masculin dit de plus précis : [apport différencié]
Ce qui est juste dans cette approche : [équité intellectuelle]
Ce qui manque : [compléments du Code]"
```

---

### BLOC 4 — 3 TAKEAWAYS (2-3 min)
```
"Voilà les 3 choses à retenir de cette vidéo :

1. [Takeaway 1 — formulé comme une vérité concrète]
2. [Takeaway 2 — formulé comme une règle de vie]
3. [Takeaway 3 — formulé comme une action]

Celui qui change tout pour la plupart des hommes que j'accompagne : le numéro [X]."
```

---

### BLOC 5 — CTA (1 min)
```
"Si cette vidéo t'a apporté quelque chose — j'ai une demande.

Laisse un commentaire avec 1 chose que tu vas faire différemment cette semaine.
Pas pour moi — pour toi. Écrire un engagement le rend 3 fois plus probable.

Et si tu veux aller plus loin sur [pilier travaillé] —
le Quiz des 12 Piliers est dans la description.
Il t'aide à savoir exactement où tu en es.

À la prochaine."
```

---

## RÈGLES DE COMPORTEMENT

- **Voix Prince Johann** — appliquer `skill-voix-prince-johann`
- **Pas de générique avant la valeur** — jamais
- **Maximum 3 sous-parties dans l'analyse** — au-delà, l'attention se perd
- **Le CTA demande 1 action dans les commentaires** — engagement de l'audience
- **Durée visée : 8-15 min** — pas en dessous de 8 (pas assez de valeur), pas au-dessus de 15 (perte d'attention)
