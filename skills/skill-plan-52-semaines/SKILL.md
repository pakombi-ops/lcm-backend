# Plan 52 Semaines — Programme Code Masculin

## ARCHITECTURE DU PROGRAMME

Le programme 52 semaines est la version la plus complète du Code Masculin. 1 module par semaine, sur 12 mois, suivant la progression des 3 phases.

```
PHASE 1 — FONDATION (Semaines 1-17)
Piliers : Force Physique, Discipline, Présence, Stoïcisme
"Sans fondation solide, le reste s'effondre."

PHASE 2 — IDENTITÉ (Semaines 18-34)
Piliers : Leadership, But, Honneur, Authenticité
"Savoir qui tu es et où tu vas."

PHASE 3 — IMPACT (Semaines 35-52)
Piliers : Vulnérabilité, Générosité, Courage, Héritage
"Créer un impact qui dure."
```

---

## STRUCTURE DE CHAQUE MODULE

Chaque module suit le même format :

| Composant | Description |
|---|---|
| Vidéo d'enseignement | 15-25 min — concept + histoire + application |
| PDF guide | Résumé + exercices pratiques + journal de réflexion |
| Défi hebdomadaire | 1 action concrète à accomplir dans la semaine |
| Audio guidé | Méditation / visualisation liée au pilier (optionnel) |

## PROTOCOLE D'ORIENTATION

### Si l'homme démarre le programme

```
Questions d'orientation :
1. "As-tu fait le diagnostic 12 piliers ?"
   → Si non : orienter vers skill-diagnostic-12-piliers d'abord
   → Si oui : utiliser les scores pour déterminer le point de départ

2. "Quel est ton score global sur 120 ?"
   → 90+ : commencer Module 1, rythme standard
   → 60-89 : commencer Module 1, renforcer les piliers faibles en parallèle
   → < 60 : commencer Module 1 mais doubler le temps sur Phase 1

3. "Combien de temps peux-tu consacrer par semaine au programme ?"
   → 3h+ : rythme standard (1 module/semaine)
   → 1-2h : rythme adapté (1 module/2 semaines)
   → < 1h : version essentielle (vidéo + 1 exercice clé uniquement)
```

### Si l'homme est déjà dans le programme

```
"Où en es-tu dans le programme ?"
→ Identifier le module actuel
→ Résumer ce qui a été fait
→ Préparer le prochain module avec : objectif + défi + exercice clé
```

### Si l'homme a décroché

```
"Tu as manqué combien de semaines ?"
→ 1-2 semaines : reprendre où il s'est arrêté, pas de culpabilité
→ 3-4 semaines : refaire le dernier module avant de continuer
→ 1 mois+ : refaire le bilan de phase précédent, puis reprendre

"Ce qui a causé le décrochage — c'était quoi ?"
→ Identifier et adresser le blocage avant de reprendre
```

## RÈGLES DE COMPORTEMENT

- **Respecter la séquence** — ne jamais recommander de sauter la Phase 1 même si l'homme veut aller directement à l'Impact
- **Adapter le rythme sans briser la structure** — ralentir si nécessaire, jamais sauter
- **Connecter chaque module à la vie réelle de l'homme** — ses situations concrètes, pas des exemples abstraits
- **Voix Prince Johann obligatoire** — appliquer `skill-voix-prince-johann`
- **Célébrer les bilans de phase** — Module 17, 35 et 52 sont des moments de reconnaissance, pas juste des évaluations
